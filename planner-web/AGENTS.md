# AGENTS.md

## Commands
- Use `pnpm`; the repo has `pnpm-lock.yaml` and no other lockfile.
- Dev server: `pnpm dev`.
- Production build: `pnpm build` runs `tsc -b` before `vite build`.
- Focused checks: `pnpm lint`, `pnpm typecheck`, and `pnpm format`.
- `pnpm format` runs `eslint . --fix` first, so import sorting comes from ESLint before Prettier writes non-generated TS/TSX files.
- No test script or test runner is configured in `package.json`.

## Generated Code
- Do not edit `src/routeTree.gen.ts`; it is generated by `@tanstack/router-plugin/vite` from files under `src/routes`.
- Do not hand-edit `src/api/generated/**`; regenerate with `pnpm api:generate` or `pnpm api:generate:watch`.
- API generation expects the backend OpenAPI JSON at `http://127.0.0.1:3000/api/v1/docs-json`.

## App Wiring
- Entry point is `src/main.tsx`; it installs `QueryClientProvider`, `ThemeProvider`, and `RouterProvider`.
- TanStack Router context includes `queryClient` from `src/api/query-client.ts`; route loaders use `context.queryClient.ensureQueryData(planQueries...)`, and route components consume the same query with `useSuspenseQuery` or `Route.useLoaderData` when no cache subscription is needed.
- API base URL comes from `VITE_API_BASE_URL`, defaulting to `http://127.0.0.1:3000` in `src/api/env.ts`.
- Import aliases use `@/*` for `src/*`; TypeScript allows explicit `.ts`/`.tsx` import extensions.

## Conventions
- shadcn/ui is configured by `components.json`: components go under `src/components`, UI aliases are `@/components/ui`, and utilities are `@/lib/utils`.
- Prefer multi-step forms for non-trivial create/import flows, and persist non-sensitive form state in the URL with `nuqs`.
- Do not persist secrets, tokens, credentials, or sensitive financial details in URL params.
- Do not duplicate loader-backed data with `useQuery` plus manual loading/error branches; normalize controlled form inputs to strings with shared utilities such as `readText` to avoid null value crashes.
- Prettier uses no semicolons, double quotes, 2 spaces, 80 columns, and `prettier-plugin-tailwindcss` with Tailwind functions `cn` and `cva`.
- ESLint ignores `dist` and `src/api/generated`; route files disable `react-refresh/only-export-components` because TanStack Router route modules export `Route`.
